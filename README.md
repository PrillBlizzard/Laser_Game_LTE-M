# Laser_Game_NR-_Codium
Ce projet à été réalisé dans le cadre de mon projet de fin d'étude (PFE) de ma dernière année d'école d'ingénieur.

## Description
C'est un projet de dévellopement d'un équipement de Laser Game tirant en infrarouge et où l'équipement envoie des données sur un serveur via LTE-M.
Le prototype software est utilisé sur une carte de dévellopement nRF9151-DK.

## Arborescence 
Vous trouverez dans le dossier racine le dossier "Prog_Carte_Globale" correspondant au sur-dossier de l'application créer par l'extension nrfConnect pour VScode.
Vous trouverez aussi dans ce dossier mon compte-rendu de PFE : essentiel pour comprendre ce projet.
